Objective:
-------------
Learn about a simple DFT-SCF algorithm and optimize it at various levels using high-performance computing techniques such as vectorization, sparse matrices, multiprocessing, Numba, PyPy, Cython, and OpenMP.

Methods:
-------------
1. **Before Optimization** – baseline Python loops
2. **Vectorized** – Numpy vectorization
3. **Sparse Laplace** – using scipy.sparse for the Laplacian
4. **PyPy** – runtime improvement via alternative Python interpreter
5. **Numba** – JIT compilation of critical functions
6. **Multiprocessing** – parallel computation for Hartree potential
7. **OpenMP** – parallelization using Cython + OpenMP

Observations and Conclusions:
------------------------------
1. Eigenvalue solution dominated the runtime in unoptimized Python loops.  
2. Vectorization alone did not significantly reduce total runtime.  
3. Sparse Laplace matrices reduced Hamiltonian and eigenvalue computation drastically.  
4. Numba JIT and OpenMP provided the greatest speedup (~20×) for Hartree potential computation.  
5. PyPy showed moderate improvements, but not as effective as Cython + OpenMP.  
6. Multiprocessing and OpenMP reduced Hartree potential time, with OpenMP being more efficient for shared-memory parallelism.

#Among all the optimization techniques applied, the combination of Sparse Matrices with OpenMP parallelization (or Numba JIT for pure Python) provided the greatest runtime improvement.
